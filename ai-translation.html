AI Translation

This label indicates that the content was originally written by a human but translated using AI without altering the intent, meaning or structure of the original text.

Dos and Don'ts

Dos

  • Use when the translation is AI-generated but based on human-authored source text.
  • Proofread the translation to ensure factual and contextual accuracy.
  • Keep the source content available for reference if possible.

Don'ts

  • Don’t use this label if AI also rewrote or expanded the text.
  • Don’t mark content as “AI Translation” if the source was AI-generated.
  • Don’t use this label as a substitute for proper human review of technical translations.

Examples

  • OK: A human-written policy document translated into English using AI.
  • OK: A product description translated automatically but reviewed by an editor.
  • Not OK: AI rewriting parts of the text — use “AI Support”.
